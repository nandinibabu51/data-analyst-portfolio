## Project Overview
This project analyzes retail sales data to understand overall sales performance, customer behavior, and product trends using Excel.

## Tools Used
- Microsoft Excel

## Dataset Description
The dataset contains sales transactions including:
- Product category and sub-category
- City and branch
- Customer type and gender
- Payment method
- Sales amount, tax, and quantity

## What I Did
- Cleaned raw sales data and handled missing values
- Created pivot tables to summarize sales, tax, and quantity
- Built an interactive Excel dashboard with KPIs
- Analyzed product-wise, city-wise, and payment-wise performance

## Key Insights
- Certain cities and branches contribute significantly higher sales
- Fashion and electronic accessories are top-performing categories
- Credit card and cash are the most used payment methods
- Member customers generate higher total sales than normal customers

## Business Impact
This analysis helps businesses understand sales patterns, optimize product focus, and improve customer targeting strategies.
